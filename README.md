# QWorld-QC-Certification-Course
QWorld NYUAD 23' Hackathon QC Certification Course

A combined version of the Bronze and Silver QC courses, the class consisted of 9 Quizzes and 2 short Projects that provided an overview of quantum computational basics, the Quantum Fourier Transform, Grover's Algorithm, Shor's Algorithm, and Order Search Algorithm.
